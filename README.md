# 📊 Customer Behaviour Analysis

## 📌 Project Overview
This is an end-to-end Data Analytics project focused on analyzing customer purchasing behavior using Excel dataset.

The project covers the complete data pipeline:
- Data Cleaning using Python
- Business Insights using MySQL
- Data Visualization using Power BI

---

## 🛠️ Tools & Technologies
- Python (Pandas)
- MySQL
- Power BI
- Excel Dataset

---

## 🔄 Project Workflow
1. **Data Cleaning (Python)**
   - Handled missing values
   - Data transformation
   - Data formatting and preprocessing

2. **Data Analysis (MySQL)**
   - Wrote SQL queries to solve business problems
   - Performed aggregations
   - Extracted meaningful insights

3. **Dashboard (Power BI)**
   - Created interactive dashboard
   - Used filters (Category, Gender, Discount, Subscription)
   - Visualized KPIs and trends

---

## 📊 Dashboard Insights

### 🔹 Key Metrics
- Total Customers: 5K  
- Unique Items: 30  
- Average Rating: 3.62  
- Total Spend: 994K  
- Average Spend: 198.84  

### 🔹 Business Insights
- Electronics category generated highest revenue
- Top locations: New York, Los Angeles
- Most used payment methods: Debit Card, Cash
- Seasonal trend shows higher sales in Winter & Summer
- Top products: Headphones, Phone, Bag
